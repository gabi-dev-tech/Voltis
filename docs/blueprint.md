# **App Name**: EduCyber Dashboard

## Core Features:

- Student Progress Metrics: Display key progress metrics like completed courses, grades, and study time.
- Active Courses Overview: List all active courses with visual progress bars indicating completion percentage.
- Performance Charts: Generate interactive charts displaying weekly and monthly performance trends.
- Actionable Task List: Show upcoming deadlines, recent achievements, and study time allocation at a glance.
- Quick Action Buttons: Provide immediate access to class lectures, assignment submissions, and grade overviews.

## Style Guidelines:

- Primary color: Cyan (#00E5FF) for a bright, digital feel, drawing from the "CyberNeonUI" design.
- Background color: Dark gray (#0A0E14), a desaturated hue close to the primary to reinforce the dark theme, but allows for contrast with the UI elements
- Accent color: Magenta (#FF00AA) to provide a visual pop that is analogous to the cyan and adds to the cyberpunk aesthetic.
- Headline font: 'Orbitron' sans-serif for headings, mirroring the given design system's preference for a modern and techy feel.
- Body font: 'Inter' sans-serif for body text, ensuring readability with a modern aesthetic.
- Use a card-based layout following the provided design system, with frosted glass panels, neon glows, and sharp geometry + rounded corners
- Incorporate subtle neon glow animations on hover for cards and buttons, enhancing interactivity and the cyberpunk feel.
